# Job Scheduling Optimizer

An interactive Streamlit app that compares **Greedy**, **Dynamic Programming**, and **Backtracking** algorithms for the Job Scheduling problem.

## Features
- CSV upload or manual job input
- Gantt chart visualization
- Profit comparison graph
- Algorithm performance comparison

##  Algorithms Used
- **Greedy (Shortest Job First)**
- **Dynamic Programming (Weighted Interval Scheduling)**
- **Backtracking (Brute Force)**

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run job_scheduling_app.py
